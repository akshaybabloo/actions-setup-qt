Create a setup content in setup-qt.ts file to setup qt environment for windows, macos and linux using the following online installer, you need to download them first.

- For windows x64 use https://download.qt.io/official_releases/online_installers/qt-online-installer-windows-x64-online.exe
- For windows arm use https://download.qt.io/official_releases/online_installers/qt-online-installer-windows-arm64-online.exe
- For macos use https://download.qt.io/official_releases/online_installers/qt-online-installer-mac-x64-online.dmg (
    dont forget to mount it first and use /Volumes/qt-unified-macOS-x64-[Qt Online Installer version]-online/qt-unified-macOS-x64-[Qt Online Installer version]-online.app/Contents/MacOS/qt-unified-macOS-x64-[Qt Online Installer version]-online
    to run the installer)
- For linux x64 use https://download.qt.io/official_releases/online_installers/qt-online-installer-linux-x64-online.run
- For linux arm use https://download.qt.io/official_releases/online_installers/qt-online-installer-linux-arm64-online.run


Each of these installers should be run with the following parameters - "install qt6.10.0-full-dev --email <username> --password <password> --root $HOME/Qt --accept-licenses --default-answer --confirm-command

Don't forget to export PATH=$HOME/Qt/<version>/<compiler>/bin:$PATH after installation.

Don't forget to cache it after installation.
